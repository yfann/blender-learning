




## tab



<!-- edit -->
<!-- 选择物体 -->
+ `tab` 编辑模式
+ `x` 删除,熔并
+ `shift + d` 复制物体并进入grab模式
+ `F9` 上一步微调面板
+ `f`
    + 选中线，填充面
    + 选中点，连接线
+ `i` 内插面
+ `e` 挤出(曲线选择顶点)
    + `z` 按z轴移动

<!-- 面操作 -->
+ 选择面(edit模式下)
    + `k` 手动分割面
        + `ctrl + k` 吸附切割
        + `enter/space` 确认
        + `Esc` 取消

<!-- 顶点 -->
<!-- 选中定点 -->
+ `shift + v` 滑移定点
+ `m` 选中多个定点合并 
+ `j` 多个顶点连成线，分割平面

<!-- 曲线 -->
<!-- 选择顶点 -->
+ `x` 删除
+ `e` 挤出，添加新点
+ `v` 选择顶点属性
    + 自动平滑
+ `f` 曲线闭合
