
## 修改器
+ select -> 右侧扳手图标 -> add modifier
+ `ctrl A` 应用修改器

## modifiers
+ subdivision
    + 用于增加模型的细分细节和光滑度。它通过对模型的几何结构进行分割和插值来实现这一目的。

+ solidify
    + 平面加厚度

+ bevel

+ Boolean
    + 不同物体重叠部分去交集，差集，并集
